

# camTracking
Tracking only using cameras with a prior lidar map. We pose the problem on a factor graph.

# Usage
```
# To run with factor graph
python main_alt.py
# To run plain VO
python vo_tracking.py
# To run CMRNet based tracking
python cmr_tracking.py
```
